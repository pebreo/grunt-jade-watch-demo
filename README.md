INTRODUCTION
---
A demo on how to generate HTML files with Jade using Grunt.

INSTALLATION
---
a) Make sure you have npm installed
b) Make sure you have Gruntfile.js and package.json
1) npm install grunt --save-dev
2) grunt watch
